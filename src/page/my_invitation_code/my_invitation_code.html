什么是邀请码？
邀请码规则

1. 每个大V店会员都有一个专属邀请码，您可以将您的邀请码分享给身边的非会员朋友；

2. 邀请码为6位数字+字母组合，您的大V店账户手机号也可作为邀请码使用；

3. 好友选定您作为邀请人后，TA在大V店APP及果敢时代大V店公众号内都将访问您的店铺；

4. 一个用户只能有一个邀请人，添加邀请人后可在7天内更换一次；更换后不可再修改。
